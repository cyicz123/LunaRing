# 更新日志（月环）

## 1.0.3 (2026-02-13)
- 新增应用内更新能力：可在设置页检查 GitHub 最新版本并提示更新
- 新增 APK 下载与安装流程：支持下载进度显示并拉起系统安装器完成更新
- 新增 GitHub Actions 自动打包发布流程：基于版本 Tag 自动构建并上传 Release APK

## 1.0.2 (2026-02-12)
- 新增通知权限引导弹窗（与应用风格一致），仅在用户选择开启通知时触发系统权限申请
- 设置页手动开启通知时，若未授予权限会先引导申请，避免“开关开启但无效”
- 未授予通知权限时不再调度 WorkManager 通知任务，避免误导

## 1.0.1 (2026-02-12)
- 应用名称改为“月环”
- 启动图标替换为自定义图标（`月环.png`）
